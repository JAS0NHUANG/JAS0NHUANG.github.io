SASS 一般環境使用

SASS 現在也有 NPM 的包，只要 npm install -g sass 就可以安裝在 Node 的環境下使用了。
轉換 .scss 或 .sass 檔案也很簡單，就是 sass INPUT.sass OUTPUT.css 就可以了，然後如果要自動監控 sass 檔在指令裡加 –watch 的選項就可以了。
